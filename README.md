
# Shape Up - ترجمه فارسی

این ریپو ترجمه فارسی کتاب [Shape Up](https://basecamp.com/shapeup) نوشته Ryan Singer است. کتاب روشی عملی برای شیپینگ، شرط‌بندی و عرضه کردن پروژه‌های نرم‌افزاری در چرخه‌های شش‌هفته‌ای توضیح می‌دهد.

## اطلاعات

- نویسنده: Ryan Singer
- مترجم: [محمد تکلو](https://profile.ir/takalloo.mohamad/about)
- متن اصلی: [basecamp.com/shapeup](https://basecamp.com/shapeup)
- سایت ترجمه: [shapeupbook.ir](http://shapeupbook.ir)
- وضعیت: ترجمه فارسی کامل کتاب، شامل مقدمه‌ها، فصل‌های ۱ تا ۱۵، نتیجه‌گیری، پیوست‌ها، واژه‌نامه و درباره نویسنده

حق نشر متن اصلی متعلق به 37signals/Basecamp است. ترجمه فارسی با مجوز صاحب ریپو برای انتشار عمومی تکمیل شده است.

## ساختار ترجمه

- `docs/partone/`: شیپینگ، تعیین مرزها، پیدا کردن عناصر، ریسک‌ها و نوشتن پیچ
- `docs/parttwo/`: شرط‌بندی، میز شرط‌بندی و انتخاب شرط‌های چرخه بعدی
- `docs/partthree/`: ساختن، اسکوپ‌ها، نمودار تپه‌ای، تصمیم توقف و عبور از پروژه
- `docs/appendices/`: پیوست‌های اجرایی، تطبیق با اندازه تیم، شروع شیپ‌آپ، واژه‌نامه و درباره نویسنده

## تصاویر

تصاویر و نمودارهای کتاب از نسخه رسمی دریافت شده و به‌صورت محلی زیر `docs/assets/images/shapeup/` نگهداری می‌شوند تا سایت هنگام build به تصاویر خارجی وابسته نباشد.

## اجرای محلی

```bash
mkdocs serve
```

برای ساخت نسخه نهایی:

```bash
mkdocs build --strict
```
